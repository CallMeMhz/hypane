# 待办事项管理

## 任务

管理用户的待办事项，包括添加、完成、提醒等。

## 操作

### 添加待办
用户说 "帮我记住xxx" 或 "添加待办 xxx"：
1. 读取 dashboard.json
2. 找到 type=todo 的卡片（没有则创建）
3. 在 items 中添加新条目
4. 保存

### 完成待办
用户说 "完成了 xxx" 或 "xxx 做完了"：
1. 找到匹配的 todo item
2. 设置 done: true
3. 保存

### 每日提醒
定时执行时：
1. 检查 dueDate 是今天或已过期的未完成项
2. 如果有，确保 todo 卡片在前面位置

## 待办格式

```json
{
  "id": "唯一ID",
  "text": "待办内容",
  "done": false,
  "dueDate": "2026-02-20"  // 可选
}
```
